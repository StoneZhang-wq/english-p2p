声网联调（免预约）

当前后端 不会校验预约或配对；房间页通过 URL 指定频道与 UID 即可向 /api/agora/rtc-token 取 Token 入会。正式产品上线前建议在服务端为 Token 增加「登录 + 属于该场次 pair」校验。

📱 两台设备（或两个浏览器窗口）进入同一 channel、不同 uid，才能互相听到声音。请允许麦克风权限；HTTPS 站点才符合浏览器媒体权限要求。

测试用频道名（可改 URL 中的 channel）： cloud-dev

  • 用户 A（uid 10001）： 打开房间
  • 用户 B（uid 10002）： 打开房间

若入会失败或报错 CAN_NOT_GET_GATEWAY_SERVER / no active status：

  • Railway 环境变量 AGORA_APP_ID、AGORA_APP_CERTIFICATE 须来自同一声网控制台项目，无首尾空格；证书需与「主要证书」一致。
  • 控制台中项目已启用、RTC 未停用；新创建项目若未激活也会导致网关无可用状态（类似 INVALID_VENDOR_KEY 类问题）。
  • 国内访问国际区 App 或线路异常时，可在房间 URL 增加 &agoraArea=CHINA 或 &agoraArea=GLOBAL（在 createClient 前由 SDK setArea 指定区域，见 setArea）。
  • 排除本机 DNS/防火墙：可尝试更换网络或 DNS（声网文档建议国内 114.114.114.114、海外 8.8.8.8），并放行 UDP。

同时请确认 CORS_ORIGINS 包含当前访问域名（如 https://english-p2p-production.up.railway.app）。
